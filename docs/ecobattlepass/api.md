## Source Code

The source code can be found [here](https://github.com/Auxilor/EcoBattlepass).

## API

Add this to your build.gradle.kts:

```kts
repositories {
    maven("https://repo.auxilor.io/repository/maven-public/")
}

dependencies {
    compileOnly("com.willfp:EcoBattlepass:<version>")
}
```

The latest version available on the repo can be found [here](https://github.com/Auxilor/EcoBattlepass/tags)
