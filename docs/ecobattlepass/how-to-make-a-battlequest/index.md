## What is a BattleQuest?

A BattleQuest is a specific task that players can complete to earn Battle Pass experience. Each quest belongs to a category, and each category belongs to a battle pass. For example, you could have a "Combat" category with quests like "Kill 10 Zombies" or "Kill 5 Players", and a "Building" category with quests like "Place 20 Blocks" or "Craft a Pickaxe".

## How to make a BattleQuest

To create a BattleQuest, you need to configure three things: A Category, a Quest, and a Task.

The Category is the highest level and contains multiple Quests. Each Quest contains multiple Tasks. Tasks are the specific objectives that players need to complete to earn experience.
