## What is EcoCollections?

EcoCollections is a fully customizable and feature-rich Collections plugin designed to reward players for interacting with the world. Players progress through tiers across collections in categories like Mining, Combat, Farming, Fishing, and Foraging, earning rewards as they go.

With EcoCollections, you can create collections for any item, mob, or resource, each with unique tier requirements, rewards, and effects—ideal for providing long-term progression and replayability.

## Key Features

### Extensive Effects System

Build dynamic collection rewards using over 200 effects, 100 conditions, and 100 triggers, powered by libreforge. From granting potion effects to running commands, enchanting items, applying multipliers, and more—there are no limits.

### Plugin Integrations

Supports deep integration with other eco plugins like EcoSkills and EcoEnchants, as well as compatibility with other popular plugins such as AxEnvoy, McMMO, and more—ensuring smooth interaction across your server's ecosystem.

### Customizable GUIs

Create fully custom menus with support for custom items from ItemsAdder, Oraxen, Nexo, or any item using CustomModelData. Design a collections interface that matches your server's branding and theme.
