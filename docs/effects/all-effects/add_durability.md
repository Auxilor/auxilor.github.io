# `add_durability`

Increase the max durability of an item

> [!danger] Triggered Effect

> [!warning] Requires 1.21+

> [!Example]
> ```yaml
> effects:
>- id: add_durability
>   args:
>     durability: 100 # The durability to add (or subtract)
  ...other config (eg triggers, filters, mutators, etc)
