# `give_battlepass_xp`
#### Triggered Effect

Give battlepass experience points

**Requires xBattlepass**
# Effect Syntax
```yaml
- id: give_battlepass_xp
  args:
    amount: 100 # The amount of xp to give
    battlepass: example # The battlepass ID
  ...other config (eg triggers, filters, mutators, etc)
```
