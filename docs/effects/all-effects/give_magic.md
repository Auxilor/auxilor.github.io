# `give_magic`

Add / subtract magic

> [!danger] Triggered Effect

> [!warning] Requires EcoSkills

> [!Example]
> ```yaml
> effects:
> - id: give_magic
>   args:
>     type: mana # The point to add/subtract
>     amount: 15 # The amount to add/subtract
>   ...other config (eg triggers, filters, mutators, etc)
> ```
