# `give_mcmmo_xp`
:::dangerTriggered Effect
:::

Gives experience points for a certain skill

**Requires mcMMO**

# Effect Syntax
```yaml
- id: give_mcmmo_xp
  args:
    amount: 100 # The amount of xp to give
    skill: exploration # The skill to give the xp for
  ...other config (eg triggers, filters, mutators, etc)
```
