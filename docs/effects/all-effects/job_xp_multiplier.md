# `job_xp_multiplier`
:::infoPermanent Effect
:::

Multiplies job xp gain

**Requires EcoJobs**

# Effect Syntax
```yaml
- id: job_xp_multiplier
  args:
    multiplier: 1.5 # The experience multiplier
    jobs: # The list of jobs to multiply xp for. If removed, it will multiply all jobs.
      - miner
      - builder 
```
