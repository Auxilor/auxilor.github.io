# `jobs_xp_multiplier`
#### Permanent Effect

Multiplies xp gain from jobs

**Requires Jobs Reborn**

# Example Config
```yaml
- id: jobs_xp_multiplier
  args:
    multiplier: 1.5 # The experience multiplier
    jobs: # The list of jobs to multiply xp for. If removed, it will multiply all jobs.
      - miner
      - fisherman
```
