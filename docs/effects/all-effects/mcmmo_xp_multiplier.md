# `mcmmo_xp_multiplier`
:::infoPermanent Effect
:::

Multiplies mcMMO skill xp gain

**Requires mcMMO**

# Effect Syntax
```yaml
- id: mcmmo_xp_multiplier
  args:
    multiplier: 1.5 # The experience multiplier
    skills: # The list of skills to multiply xp for. If removed, it will multiply all skills.
      - mining
      - combat 
```
