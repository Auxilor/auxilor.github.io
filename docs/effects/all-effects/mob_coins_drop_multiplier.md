# `mob_coins_drop_multiplier`
:::infoPermanent Effect
This effect is permanent and does not require a trigger.
:::

Multiplies the mobcoins dropped


:::dangerRequires:
UltimateMobCoins
:::

# Effect Syntax
```yaml
- id: mob_coins_drop_multiplier
  args:
    multiplier: 1.5 # The drop multiplier
```
