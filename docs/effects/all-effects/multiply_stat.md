# `multiply_stat`
:::dangerPermanent Effect
This effect is permanent and does not require a trigger.
:::

Multiplies a stat by a specific value

:::infoRequires:
EcoSkills
:::

# Effect Syntax
```yaml
- id: multiply_stat
  args:
    stat: speed # The name of the stat
    multiplier: 0.75 # The amount to multiply the stat by
```
