# `oxygen_bonus`
:::dangerPermanent Effect
This effect is permanent and does not require a trigger.
:::

Adds oxygen bonus

# Effect Syntax
```yaml
- id: oxygen_bonus
  args:
    amount: 1.5 # The amount of oxygen bonus to add
```
:::tip

It's advised you read into this attribute [here](https://minecraft.wiki/w/Attribute) to understand what it does.

:::
