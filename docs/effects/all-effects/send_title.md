# `send_title`

Send a title/subtitle to the player

> [!danger] Triggered Effect

> [!example]
> ```yaml
> - id: send_title
>   args:
>     title: "&aCongratulations %player%!" # The title
>     subtitle: "&3You won $1000" # The subtitle
>   ...other config (eg triggers, filters, mutators, etc)
> ```
