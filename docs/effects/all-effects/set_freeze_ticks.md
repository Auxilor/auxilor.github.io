# `set_freeze_ticks`

Sets the victims freeze ticks (frost / powdered snow effect)

> [!danger] Triggered Effect

> [!example]
> ```yaml
> - id: set_freeze_ticks
>   args:
>     ticks: 20 # The ticks to set (How long the victim has been in snow)
>   ...other config (eg triggers, filters, mutators, etc)
> ```
