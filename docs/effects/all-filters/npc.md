# `npc`

Require a certain NPC

:::warningRequires:
Citizens || FancyNpcs
:::

# Filter Syntax

**Citizens**
```yaml
filters:
  npc: # The Citizens NPC IDs
    - 1
    - 15
```

**FancyNpcs**
```yaml
filters:
  npc: # The FancyNpcs NPC IDs (UUID from npcs.yml)
    - 5ccc42ba-82ff-4ca3-a1a8-408bf8187170
```
