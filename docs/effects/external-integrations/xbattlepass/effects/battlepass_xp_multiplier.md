# `battlepass_xp_multiplier`
:::infoPermanent Effect
This effect is permanent and does not require a trigger.
:::

Multiplies incoming battlepass xp gain

:::dangerRequires:
xBattlepass
:::
# Example Config
```yaml
- id: battlepassxp_multiplier
  args:
    multiplier: 2.05 # The xp multiplier
    battlepasses:
      - battlepass
```
